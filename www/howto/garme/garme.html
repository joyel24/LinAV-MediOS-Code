HOW-TO build GARME (The AV3xx emulator) ?

Answer: Try to follow the hereunder guide!



These guide has been made for Linux users, but it should be usable in Cygwin with slight modification.



  1. Prerequisite: You need:

  • a jvm (included in the jre or jdk)

  • ant

  1. We get the sources from CVS.

    cd ~/linav
    cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/garme login
    cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/garme co garme

    After the first line, you will be prompted for a password. Simply press Return.

  2. Let's compile !

    cd garme
    ant

  3. We can test the emulator. Get this file and put it in your garme directory.


    ./run hello.bin

  4. On the debug console, you should see “Hello world” . Fine it's working well !
